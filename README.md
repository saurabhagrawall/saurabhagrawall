## Featured Project
### AI Powered API Contract Monitor

A full stack system that detects breaking API changes across microservices, analyzes service dependencies, and suggests migration strategies using AI.

Built to solve real problems observed while working on production systems, where small API changes could cause cascading failures across services.

Key highlights:

• Microservices architecture built with Java, Spring Boot, and PostgreSQL  
• Integrated LLM based analysis to explain API changes and potential impact  
• React dashboard to visualize dependencies and version drift  
• Automated detection of breaking changes across service contracts  
• Database per service design for isolation and resilience  
• OpenAPI based contract tracking and validation  

Focus areas: system reliability, developer tooling, and scalable backend architecture.
